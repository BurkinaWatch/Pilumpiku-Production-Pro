 SITE WEB PILIMPIKU PRODUCTION 

## LOGO & IDENTITÉ VISUELLE

Le logo officiel de Pilimpiku Production est un **papillon monarque** (orange doré, brun chocolat, noir) inscrit dans un **cadre ovale**, avec les initiales **"PP"** gravées en filigrane dans les ailes.

**Signification à exploiter dans le design :**
> Le papillon = métamorphose, liberté, voyage des histoires africaines vers le monde.

**Utilisation du logo dans le code :**
```jsx
// Placer le fichier logo dans /public/images/logo-pilimpiku.png
// Utilisation dans la Navbar :
<img src="/images/logo-pilimpiku.png" alt="Pilimpiku Production" className="h-14 w-auto" />
```

**Déclinaisons du logo selon le contexte :**
- Navbar sur fond sombre → logo original (fond transparent)
- Footer → version légèrement désaturée (opacity 0.85)
- Favicon → papillon seul, sans cadre ovale, recadré carré

## PALETTE DE COULEURS — MISE À JOUR (tirée du logo)

```css
/* Couleurs extraites du logo papillon */
--orange-aile: #E8921A       /* Orange vif des ailes */
--brun-contour: #4A1F0A      /* Brun profond des bordures */
--noir-corps: #1A0A00        /* Noir chaud du corps */
--blanc-point: #FFFFFF       /* Points blancs sur les ailes */

/* Couleurs cinéma africain */
--noir-cinema: #0A0A0A
--noir-doux: #111111
--or-premium: #C9A84C
--or-chaud: #E8921A          /* ← reprend l'orange du logo */
--terre: #8B4513
--ocre: #D4783A
--rouge-profond: #7A1500
--blanc-casse: #F5F0E8
--gris-cinema: #2A2A2A
```

**Règle d'or :** L'orange du papillon (`#E8921A`) remplace le doré générique partout où il y a un accent de couleur. Cela crée une cohérence totale entre le logo et l'interface.

## CONTEXTE & IDENTITÉ

Tu es un développeur senior spécialisé en web design cinématographique. Tu dois créer le site officiel de **Pilimpiku Production**, société de production cinématographique africaine basée à **Ouagadougou, Burkina Faso**.

**Positionnement de la marque :**
- Références visuelles : A24, MUBI, Arte Cinéma, Criterion Collection
- Ton : prestige international + authenticité africaine contemporaine
- Symbole : le papillon comme métaphore de la transformation des histoires
- Promesse : *"Telling African Stories to the World"*

---

## STACK TECHNIQUE

```
Framework  : React + Vite
Styling    : Tailwind CSS + CSS custom (effets avancés)
Animations : Framer Motion
Routing    : React Router DOM
Fonts      : Cormorant Garamond (titres) + Inter (corps)
Icons      : Lucide React
Images     : Unsplash (thème cinéma / Afrique / nature)
```

**Contraintes techniques obligatoires :**
- Responsive parfait (mobile-first)
- Lazy loading sur toutes les images
- SEO : `<title>`, `<meta description>`, Open Graph sur chaque page
- Code modulaire (1 composant = 1 fichier)
- Commentaires en français

---

## DIRECTION ARTISTIQUE — RÈGLES STRICTES

### Typographie
```
Titres H1/H2    : Cormorant Garamond Bold 700, tracking large
Sous-titres     : Cormorant Garamond Italic
Corps texte     : Inter 300/400
Labels/badges   : Inter Uppercase, letter-spacing: 0.2em
```

### Effets visuels obligatoires
| Effet | Détail |
|---|---|
| Grain cinéma | Overlay SVG noise sur tout le site, opacity 0.04 |
| Vignette | Dégradé radial sombre sur les bords des heroes |
| Parallax | Sur images de fond, max 20% déplacement |
| Hover projets | scale(1.03) + overlay orange-aile + titre remonte |
| Transitions page | Fade noir 400ms entre chaque route |
| Curseur custom | Cercle orange subtil sur desktop |
| Papillons subtils | 2-3 silhouettes traversent le hero en diagonal, opacity 0.08 |
